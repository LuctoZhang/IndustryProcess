# Chemical Industry Process Visulization #

1.speed.txt文件中的数字为动画播放速度，可以手动修改。

2.Data.java文件用于获取speed.txt文件中的动画播放速度。

3.DataSource.java文件用于连接数据库，获取相关数据。

4.industry.sql文件为数据库结构和数据

5.Play.java文件中包含main函数，为程序入口，可执行。

6.Window.java文件为程序主逻辑文件，包括画面绘制、动画流程等逻辑。

7.Box.java/Slot.java/Truck.java分别为结晶箱/缓冲槽/卡车的实体类。
